# liri-node-app

# Introduction

The Liri Node application is similar to iPhone's Siri, but allows users to query movies, songs and concert dates by your favorite artists. 

### Liri's Commands

This app can take in one of the following commands:

- concert-this
- spotify-this-song
- movie-this
- do-what-it-says
